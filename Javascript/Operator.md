# Operator
- 산술 연산자: +, -..
- 대입 연산자: =
- 논리 연산자: !, &&, ||  (<- 우선순위)
- 비교 연산자
    - ==, != (비교 시 타입 변환 허용) 
    - ===, !== (비교 시 타입 변환 허용X)
> https://github.com/getify/You-Dont-Know-JS/blob/2nd-ed/types-grammar/ch4.md#loose-equals-vs-strict-equals
```js
0 == false         //true
null == undefined  //true
1 == '1'           //true
1 === '1'          //false
```
